== Description ==

MundialdeSalsa Pro is a premium WordPress theme designed for high-impact editorial projects, music magazines, and cultural portals. With a bold "Brutalist" aesthetic, it combines raw design with extreme performance and professional features.

Key Features:
*   **Brutalist Design System**: High contrast, bold typography, and raw layouts.
*   **Performance First**: Native lazy loading, deferred scripts, and optimized CSS.
*   **Editorial Engine**: Advanced breadcrumbs, reading progress bar, and social sharing.
*   **Gutenberg Optimized**: Custom block templates and full support for the block editor.
*   **Advanced Ads Manager**: Integrated system for header and article advertisements.
*   **Maintenance Mode**: Built-in "Coming Soon" functionality.
*   **Database Optimization**: One-click cleanup tools for a lean database.

== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes > Add New.
2. Click Upload Theme and select the mundialdesalsa-pro.zip file.
3. Activate the theme.
4. Install and activate the required Redux Framework plugin.
5. Go to Panel MDS Pro to configure your site.

== Frequently Asked Questions ==

= How do I enable Dark Mode? =
The theme includes a native dark mode toggle in the header. You can also set the default skin in the Presets section of the MDS Pro Panel.

= Can I use it for other music genres? =
Absolutely. While designed with Salsa in mind, the design system is versatile for any high-energy editorial project.
