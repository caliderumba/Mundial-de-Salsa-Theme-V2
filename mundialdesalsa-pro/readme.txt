== Description ==

MundialdeSalsa Magazine Pro is a premium WordPress theme designed for salsa news, culture, and multimedia portals. It features a high-performance design, dark mode support, and advanced theme options.

== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes > Add New.
2. Click Upload Theme and select the mundialdesalsa-pro.zip file.
3. Activate the theme.
4. Install and activate the Redux Framework plugin for theme options.

== Features ==

- Dark Mode Toggle
- Live Search (AJAX)
- Infinite Scroll (AJAX)
- Social Sharing Buttons
- Breadcrumbs
- Advanced Theme Options (Redux)
- High Performance (Tailwind CSS)
